# Manual Goal Publisher

## 개요
사용자 입력을 통해 수동으로 목표점을 발행하는 노드입니다.

## 주요 기능
- 사용자 입력 좌표 처리
- 좌표 유효성 검사
- 실시간 목표점 발행

## 토픽
### 발행
- `/goal_point` (geometry_msgs/msg/Point): 목표점 정보

## 입력 형식
- "x y" 형식의 좌표 입력
- 입력 범위: -10m ~ 10m
- 'q' 입력으로 프로그램 종료

## 주요 기능
1. 좌표 입력 처리
   - 공백으로 구분된 x, y 좌표 파싱
   - 숫자 형식 검증
   - 범위 검증

2. 오류 처리
   - 잘못된 입력 형식 감지
   - 범위 초과 감지
   - 오류 메시지 출력
